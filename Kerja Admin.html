KERJA ADMIN
Website ini dibuat bertujuan untuk menganalisis hal apa saja yang berkemungkinan memiliki potensi dalam membuat kericuhan, kekacauan, kebigungan, hoax, dll. Oleh karena itu website ini merupakan website berkategori E-News, dan membahas hal-hal secara mendalam dan mendetail.
Website ini juga dibuat baru pada tahun 2024/06/14 terhitung dari saat mulainya pembuatan website ini berjalan.
Untuk kedepannya saya sebagai Admin dan juga developer Website Bravery ini berkeinginan untuk terus mengikuti apa saja masalah yang akan terjadi kedepannya. Oleh karena itu tugas dan sekaligus perkerjaan saya akan memakan waktu 1 bulanan untuk membuat suatu hasil yang signifikan. Dan oleh karena itu juga saya meminta pengertian dari para pembaca untuk sabar.
Sekian dari saya terimakasi.
